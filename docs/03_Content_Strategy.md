# Writing Style

Short paragraphs.

Short headlines.

No unnecessary technical explanations.

Use active voice.

Professional tone.

Business language.

# Headlines

# Image Strategy

One exceptional image is worth more than ten average renders.

Never upload work that lowers the overall portfolio quality.

Quality always wins over quantity.

---

# Portfolio Growth Strategy

Every new project added should make the portfolio stronger.

If it does not improve the portfolio,

do not publish it.

# Success Metric
